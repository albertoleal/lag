## Install:

```
  go get github.com/tscolari/lag
```

## Usage:

```
  lag logfile.log
```

## Navigation:

* `Q` exits
* `arrow up` moves cursor up
* `arrow down` moves cursor down
* `enter` zooms into the selected entry
* `q` zooms out from the selected entry
* `backspace` zooms out from the selected entry
